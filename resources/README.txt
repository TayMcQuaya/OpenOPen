OpenOPen
========

OpenOPen is a free and open-source text editor developed by TayMcQuaya. It provides powerful tools for creating and editing documents while remaining simple and user-friendly.

Features
--------
- Text formatting: bold, italic, underline
- Bullet and numbered lists
- Export to DOCX and PDF
- Undo/redo functionality
- Font and size customization
- Dark mode support
- Zoom in/out for better readability
- Search & replace text
- Print support
- Open and edit DOCX/RTF files
